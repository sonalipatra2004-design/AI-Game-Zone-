# 🎮 AI Game Zone

AI Game Zone is an interactive game application built using Streamlit. It provides fun and educational games with a modern user interface.

## Features

- 🖼️ Image Quiz
- 🧩 Riddle Challenge
- ✏️ Draw & Guess
- 🎨 Attractive and responsive UI
- ⚡ Fast and lightweight
- 🌐 No API required
- 💻 Fully offline

## Technologies Used

- Python
- Streamlit
- Pillow





## Future Improvements

- AI-based image recognition
- User login and leaderboard
- Multiplayer mode
- More quizzes and riddles
